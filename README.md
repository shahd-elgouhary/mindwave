# 🧠 MindWave — AI Emotional Support System

MindWave is an AI-powered emotional support platform that combines **emotion detection, music recommendation, and conversational AI** into a unified web application built with Streamlit.

It helps users understand their emotional state, receive supportive interactions, and discover personalized music based on mood and preferences.

---

## ✨ Features

### 💬 AI Chatbot
- Conversational AI assistant for emotional support
- Multiple interaction styles
- Personalized responses based on user context

### 😊 Emotion Detection
- Real-time emotion recognition (happy, sad, angry, fear, surprise, etc.)
- Webcam / image-based emotion analysis
- Tracks emotional history per user

### 🎵 Music Recommender System
- Content-based music recommendation engine
- Suggests songs based on similarity scoring
- Integrated with iTunes API for album cover retrieval
- Emotion-based music suggestions

### 🔐 Authentication System
- Secure login & registration system
- SQLite-based user management
- Session handling using Streamlit state

---
